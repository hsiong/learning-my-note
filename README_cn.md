# 学习笔记 (learning-my-note)

这是一个全面的学习笔记库，涵盖了人工智能、后端开发、前端开发、运维架构以及各种软件开发工具。其目标是帮助开发者夯实基础，并不断向系统架构师的目标进发。

[English Version](./README.md)

## 项目介绍

本项目记录了我在不同技术领域的学习历程和经验总结。内容包括但不限于：

- **人工智能 (AI)**: 深度学习、机器学习、大语言模型 (LLM)、向量数据库等。
- **后端 (Backend)**: Java, Go, SQL 优化, 大数据技术等。
- **运维与架构 (DevOps & Architect)**: 设计模式、Docker, K8s, Linux 系统管理、架构设计等。
- **前端 (Frontend)**: JavaScript, React, Vue 等。
- **工具 (Tools)**: Git, Maven, IDE 配置, 简历模板等。

- 📂 [ai/](./ai/) — 人工智能与数据科学
