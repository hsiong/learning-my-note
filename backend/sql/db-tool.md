DBeaver **支持全表搜索**，包括在所有表、指定 schema、指定数据库中搜索字段名或字段值。下面是常用的几种搜索方式：

------

## ✅ **1. 搜索表名 / 列名（元数据搜索）**

适用于查找表、字段是否存在。

**操作路径：**

```
Ctrl + Shift + F
```

或：

```
Database → Search
```

你可以搜索：

- 表名
- 列名
- 视图、索引、触发器等元数据

------

## ✅ **2. 全库全文内容搜索（搜索数据内容）**

DBeaver 也可以在**所有表**中搜索某个字段值（例如 "abc" 出现在哪些表的哪些字段内）。

**操作步骤：**

1. 在数据库导航树中**右键选中数据库**（或 schema）
2. 选择：

```
Search → Data Search
```

1. 在 "Data Search" 中输入要查找的内容
2. 选择搜索范围：
   - 所有表
   - 某些表
   - 某些字段
3. 点击 **Start Search**

DBeaver 会扫描你选定范围内每一张表与字段。

------

## ⚠️ 注意

- 全表搜索属于“全量扫描”，速度会根据数据量变慢
- 对分布式数据库或大表建议加 limit 或过滤条件
- 企业版支持更强大的全文搜索
